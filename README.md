# Editing Controls for Uno Platform/WinUI 3

The parent repo to manage Uno Platform/WinUI 3 related editing controls from LeXtudio Inc.

Actual issues are managed in each of the projects.

## Projects

| Project | Packages | Description |
| ------- | -------- | ----------- |
| [UnoEdit](https://github.com/lextudio/UnoEdit) | [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UnoEdit.svg?label=LeXtudio.UnoEdit&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UnoEdit) [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UnoEdit.TextMate.svg?label=LeXtudio.UnoEdit.TextMate&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UnoEdit.TextMate) | Cross platform code editor control. Ported from AvalonEdit (WPF). |
| [UnoRichText](https://github.com/lextudio/UnoRichText) | [![NuGet](https://img.shields.io/nuget/v/LeXtudio.RichText.svg?label=LeXtudio.RichText&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.RichText) | Cross platform rich text controls. Derived from RichTextBox (WPF).
| [UnoPropertyGrid](https://github.com/lextudio/UnoPropertyGrid) | [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UnoPropertyGrid.svg?label=LeXtudio.UnoPropertyGrid&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UnoPropertyGrid) | Cross platform property grid control. | [LeXtudio.Windows](https://github.com/lextudio/WindowsShims) | [![NuGet](https://img.shields.io/nuget/v/LeXtudio.Windows.svg?label=LeXtudio.Windows&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.Windows) | Shims to bridge WPF controls to Uno Platform/WinUI 3. |
| [Core Text APIs for Uno Platform](https://github.com/lextudio/TextCore.Uno) | [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UI.Text.Core.svg?label=LeXtudio.UI.Text.Core&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UI.Text.Core) [![NuGet](https://img.shields.io/nuget/v/LeXtudio.TextBox.svg?label=LeXtudio.TextBox&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.TextBox) | Cross platform IME integration helper and IME aware text box control. |
