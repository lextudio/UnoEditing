# Editing Controls for Uno Platform/WinUI 3

The parent repo to manage Uno Platform/WinUI 3 related editing controls from LeXtudio Inc.

Actual issues are managed in each of the projects.

## Projects

| Project | Description |
| ------- | ----------- |
| [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UnoEdit.svg?label=LeXtudio.UnoEdit&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UnoEdit) | [UnoEdit](https://github.com/lextudio/UnoEdit), cross platform code editor control. Ported from AvalonEdit (WPF). |
| [![NuGet](https://img.shields.io/nuget/v/LeXtudio.RichText.svg?label=LeXtudio.RichText&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.RichText) | [UnoRichText](https://github.com/lextudio/UnoRichText), cross platform rich text controls. Derived from RichTextBox (WPF).
| [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UnoPropertyGrid.svg?label=LeXtudio.UnoPropertyGrid&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UnoPropertyGrid) | [UnoPropertyGrid](https://github.com/lextudio/UnoPropertyGrid), cross platform property grid control. Inspired by property grid from UWP/WPF and Visual Studio. |
| [![NuGet](https://img.shields.io/nuget/v/LeXtudio.Windows.svg?label=LeXtudio.Windows&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.Windows) | [LeXtudio.Windows](https://github.com/lextudio/WindowsShims), WPF shims to bridge WPF controls to Uno Platform/WinUI 3. |
| [![NuGet](https://img.shields.io/nuget/v/LeXtudio.UI.Text.Core.svg?label=LeXtudio.UI.Text.Core&&style=flat-square)](https://www.nuget.org/packages/LeXtudio.UI.Text.Core) | [Core Text APIs for Uno Platform](https://github.com/lextudio/TextCore.Uno), cross platform IME integration helper. |
